# Unit 02 — GIS Fundamentals

## Overview

This unit introduces Geographic Information Systems (GIS) as an integration and visualization platform for geophysical data. Students learn the essential concepts of spatial data management, coordinate systems, and cartographic representation, with a focus on workflows relevant to geophysical survey planning and result presentation using QGIS.

---

## Learning Objectives

By the end of this unit, students will be able to:

- Explain the concepts of coordinate reference systems, projections, and datums
- Distinguish between vector and raster data models and their appropriate uses
- Import, manage, and visualize geophysical survey data in QGIS
- Apply basic spatial operations: clipping, reprojection, attribute queries, buffer analysis
- Interpolate point data to continuous grids (IDW, kriging)
- Produce publication-quality maps with appropriate cartographic elements

---

## Key Concepts

- Coordinate reference systems (geographic vs. projected), datums, EPSG codes
- Vector data: points, lines, polygons; attribute tables
- Raster data: grids, resolution, band operations
- Data formats common in geophysics: CSV, Shapefile, GeoTIFF, ASCII grid
- Survey layout planning in GIS
- Spatial interpolation methods: IDW, kriging, spline
- Map composition: legend, scale bar, north arrow, graticule
- QGIS interface and key plugins for geoscience workflows
