# Unit 07 — Additional Topic

## Overview

This unit covers a topic selected each semester based on current relevance, emerging methods, or specific interests of the student cohort. Past topics have included seismic refraction, self-potential methods, induced polarization, UAV photogrammetry for geoscience, and integration of machine learning in geophysical interpretation.

The topic for the current semester is announced at the beginning of the course and detailed materials are provided under the corresponding semester folder.
